## Features
* SMPS (5V)
* LDO Regulator (3.3v)
* Relay Driver										 (2 CH)
* DC Motor Speed Controller
* 0-10 V Digital Input 					 (4 CH)
* 0-10 V Analog Input 						 (4 CH)
* 4-20 mA Receiver								 (2 CH)
* Real Time Clock
* Bluetooth Low Energy 
* WiFi Module
* SD Card Module
* RS232 Interface
* RS485 Interface
* 6 Pin ICSP Headers
* Serial Programming using UART0
